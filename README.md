# Bank API

Api for our week 1 small project.

## Requirements
* User accounts.
	- Account type
	- Personal Info
	- Account number
	- Funds
	- Interest rate
	- Alerts
* User login (tentative: password security).
* Open/close account.
* Transfer between accounts (check balance for illegal withdrawals).
* Deposit.
